# 已排除的无效事件总结

## 结论

截至本文件，三轮独立事件研究均未发现能覆盖固定完整往返成本 `0.14%` 的稳定优势。它们应保留为失败基线，不得调参后直接策略化。

数据范围说明：第一轮关键位与第二轮压缩突破报告生成于 2024 数据补齐之前，前者覆盖 2025/2026、后者同样只有当时可用的 2025/2026 切片；本总结不把它们伪称为已经用新补齐的三年数据重新验证。第三轮极端动量研究已使用 2024、2025 和当前可用的 2026 数据。若未来要重新审视前两轮，必须按原冻结规则完整重跑，而不能只挑表现较好的年份或周期。

这不是“没有交易信号”，而是三种不同失败：

| 研究方向 | 失败类型 | 是否排除 |
|---|---|---|
| 关键位突破/反转 | 样本充足，但各分组成本后收益均为负 | 是 |
| 波动率压缩后突破 | A→B 转化低、B 样本只有个位数 | 是 |
| 极端动量后下一根回中轨 | A 样本充足，但指定 B 几乎从不发生且收益为负 | 是 |
| 成交量吸收后短期反转 | B 转化尚可，但 A 样本仅 12–60，四个切片仅一个成本后为正 | 是 |

## 已排除方向

### 1. 关键位突破/反转事件

定义：5m 或 15m K 线突破过去 20 根高低点，以突破方向计算固定持有期的方向收益，并按方向、4h 标签、成交量和波动率分组。

- 样本并不缺：例如 ETH 2025 有 5m 事件 `20,149` 个、15m 事件 `6,515` 个。
- 但 ETH 2025 的方向桶成本后 1h 平均收益全部为负：5m 为 `-0.1459%/-0.1511%`（BUY/SELL），15m 为 `-0.1588%/-0.1571%`。
- BTC 2025 也同样为负：5m 为 `-0.1486%/-0.1278%`，15m 为 `-0.1846%/-0.1365%`。
- 因而不能用“加上 4h、成交量或波动率过滤”解释为优势；这些预先分组也没有产生可跨标的、跨年份的成本后正结果。

结论：**排除关键位突破/反转作为下一步策略来源。**

### 2. 波动率压缩后突破事件

定义：5m ATR(14) 低于 ATR 20 期均线的 80%，价格在已收盘 1h Bollinger 中轨附近；随后 12 根 5m K 线内，实体 K 线收盘突破 1h 上下轨。

- A→B 转化率仅 `3.41%–7.20%`。
- 每个切片只有 `6–10` 个方向性 B 事件，远低于 200 样本门槛。
- B 的成本后 1h 平均收益均为负，PF 为 `0.000–0.309`（ETH 2025 为 `0.036`）。

结论：**排除“压缩后立即突破”这一事件定义。** 它既没有足够事件，也没有成本后优势。

### 3. 极端动量后下一根 K 线回到中轨

定义：RSI(14) 超过 75/低于 25，且连续两根 5m 收盘在 Bollinger 上轨/下轨之外；每段极端行情只计首次 A。B 要求下一根 5m 收盘直接回到中轨另一侧。

- A 样本充足：完整年度 BTC 为 `648–723`，ETH 为 `735–749`。
- 但 B 转化率为 `0.00%–0.50%`，六个切片全部低于 10% 硬门槛。
- A 的反向 1h 成本后平均收益均为负，PF 为 `0.536–0.820`；最接近成本覆盖的是 ETH 2024，仍为 `-0.0694%`、PF `0.820`。

结论：**排除“极端后下一根就回中轨”的均值回归定义。** 失败原因不是样本太少，而是该回归速度在 5m 上几乎不发生。

### 4. 成交量吸收后的短期反转

定义：成交量相对前 20 根均量至少 3 倍、真实波幅不超过 0.8 ATR、过去 3 根有至少 1 ATR 单边位移；按位移反方向统计收益。B 为随后 3 根内达到 0.5 ATR 的反向移动。

- A→B 转化率为 `47.83%–63.33%`，说明短小反向波动并不少见。
- 但每个年度/标的只有 `12–60` 个 A，全部低于 200 样本门槛。
- 成本后 1h 平均收益仅 ETH 2025 为正（`+0.0518%`、PF `1.242`）；BTC 2024、BTC 2025、ETH 2024 分别为 `-0.0306%`、`-0.1626%`、`-0.1584%`。
- 四个 5m 切片未全部转正，按预设停止门槛没有运行 15m。

结论：**排除当前成交量吸收反转定义。** 高 B 转化率没有转化为稳定的成本后平均收益，且样本过少。

## 关于 15 分钟周期

周期可能影响事件的噪声、样本量和成本占比，但它不是自动修复器。

- 已有的关键位研究已经同时跑过 5m 和 15m；15m 没有把成本后负收益变为正收益。
- 压缩突破和极端动量研究目前只按 5m 定义，因此不能由此断言“它们在 15m 一定无效”；只能说没有证据表明换周期会自动带来优势。
- 成交量吸收研究因 5m 四个切片未全部转正，按预先冻结的节约成本规则没有运行 15m；这表示“本轮停止”，不等于证明该事件在所有 15m 数据上必然无效。
- 下一条假设应在设计阶段同时冻结 **5m 与 15m** 两个对照版本，分别统计，不能在看到结果后只保留较好周期。

推荐验证顺序：2024 用于冻结实现和检查样本结构；2025 作为独立验证；2026 仍是部分年度，只作为额外保留观察，不用于提前宣布通过。两种周期都扣相同的固定 `0.14%` 完整往返成本。

## 下一轮可选假设

以下两个方向都不同于关键位、布林带压缩突破和 RSI/布林带极端回归；它们只是待检验的因子假设，不是策略。

### 选项 A：成交量吸收后的短期反转（单标的）

核心问题：当成交显著放大、但价格没有相应扩张时，是否说明单边主动成交被吸收，随后出现反向短期收益？

预设 5m 事件 A：

1. 当前成交量 / 前 20 根平均成交量 `>= 3`；
2. 当前真实波幅 / ATR(14) `<= 0.8`；
3. 过去 3 根的收盘价位移绝对值 `>= 1.0 * ATR(14)`；
4. 上涨位移后标记为 `SELL` 研究方向，下跌位移后标记为 `BUY`。

标签：事件后 `5m/15m/1h/4h` 的反向方向收益，扣除 0.14% 成本。分组只按方向、成交量冲击强度三分位和吸收强度三分位。

为什么值得测：成交量此前只是关键位事件的附属特征；这里它是事件本身，且加入了“高参与、低效率”的吸收条件。它可直接在现有单标的研究框架中做 5m/15m 平行对照。

### 选项 B：BTC 与 ETH 相对价格的偏离回归（双标的）

核心问题：BTC/ETH 的相对价格偏离自身短期均值后，是否存在可覆盖双腿成本的回归？

预设 5m 事件 A：

1. 同步计算 `log(BTC close / ETH close)`；
2. 计算其过去 288 根（24 小时）滚动 z-score；
3. z-score 首次上穿 `+2`：研究方向为 `SHORT BTC / LONG ETH`；首次下穿 `-2`：研究方向为 `LONG BTC / SHORT ETH`；离开阈值区后才允许下一次 A。

标签：事件后 `15m/1h/4h` 的配对收益。因为有两条腿，完整往返成本必须预先固定为 `0.28%`，不能沿用单标的的 0.14%。

为什么值得测：它研究的是两币相对定价，不是单币关键位、波动率压缩或单币 RSI 极端。代价是若研究通过，后续需要先扩展为市场中性双腿的研究与风险模型，不能直接塞进现有单标的策略引擎。

## 选择建议

优先选择 **选项 A**：数据、成本模型和风险口径都可复用，且能干净地同时检验 5m/15m 是否真有周期差异。

若你的目标是探索真正不同于单币方向交易的来源，则选择 **选项 B**；它更独立，但研究和未来执行复杂度更高。
