# 半自动人机结合回放设计

## 边界

只读取 `data/{year}/` 的本地 CSV；没有网络行情、模拟盘或实盘接口。旧自动回测代码作为失败基线保留，但不在网页显示。

## 回放状态机

`RUNNING -> AWAITING_DECISION -> RUNNING -> FINISHED`。

服务器每次响应只返回游标以前的 K 线与已发生交易。候选信号由已收盘 5m/15m K 线生成，暂停后人可选择 `BUY`、`SELL`、`SKIP`。接受方向在下一根 K 线开盘加滑点成交，使用信号 ATR 的止损和止盈；持仓期间忽略信号，双触发先止损。交易、权益和决策记录写入 `results/manual_replays/`，不入 Git。

## 图表

使用 Lightweight Charts 展示本地 5m/15m/1h K 线。候选多头标在 low 下方，候选空头标在 high 上方；鼠标悬浮展示结构化结论和理由。图表周期仅影响观看；候选入场周期只允许 5m/15m。

## 白名单

搜索仅改变信号参数。以固定 4h 方向毛收益做标签：2024 排序、2025 验证；两年平均毛收益必须大于零，每年事件 30–100。排序为辨识度、样本量接近 65、毛收益。输出 3–5 组去重参数及触发逻辑。
